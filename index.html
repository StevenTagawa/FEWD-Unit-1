Hi. I’m Steven.

I live in Sacramento, California.

About Me

I've had many interests over the years… My first experience with programming was with GeoProgrammer (and 8086 Assembly Language) for the Commodore 64 in the late 80s, followed by Visual Basic 3.0 and 4.0 for the PC in the mid-90's.

I had my own website starting back in the mid-90s, when CSS and JavaScript were new-fangled things that nobody really used yet, and everything was styled with HTML tags. Some of the projects listed on my Projects page are things that were once on that website, and hopefully will be on a new website in the future.

For most of this century I've been working in the public sector, as a little piece of a big government (currently at the California Department of Social Services), but I'm open to other opportunities as they come up.

Goals

My immediate goal is to relearn how to create websites (the modern way) using the handy tools and technologies available in the 21st century. Eventually I plan to make use of all of the following:

  • HTML
  • CSS
  • JavaScript
  • Python
  • Flask

My plan is to make several web applications implementing ideas that I have. But designing websites for other people would be cool, too.
